# 🛒 E-Commerce Sales Data Analysis Using SQL

## 📌 Project Overview

This project focuses on analyzing e-commerce sales data using SQL to extract meaningful business insights.

The analysis includes:
- Total Revenue Analysis
- Top Selling Products
- Category-wise Sales
- Customer Purchase Behavior
- Peak Shopping Hours
- Revenue Contribution Analysis
- Cumulative Revenue Growth
- Window Functions & Ranking

---

# 🛠 Tools & Technologies Used

- MySQL
- SQL Joins
- Aggregate Functions
- Window Functions
- Business Analytics
- Data Analysis

---

# 📂 Database Schema

The project uses the following tables:

1. customers
2. orders
3. order_details
4. products
5. shipments

---

# 📊 Key Business Insights

- Electronics category generated the highest revenue
- Peak shopping hours occurred between 6 PM – 9 PM
- Premium products contributed significantly to total sales
- A few products generated the majority of revenue
- Revenue showed steady growth over time
